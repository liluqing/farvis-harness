## 交互流程

<!-- 仅描述复杂流程，简单 CRUD 可省略 -->

1. 进入页面 → 调用列表 API → 渲染表格
2. 点击新增 → 打开弹窗 → 表单初始化为空
3. 点击编辑 → 调用详情 API → 填充表单 → 打开弹窗
4. 提交表单 → 实时校验 → 调用 API → 成功后关闭弹窗 + 刷新列表
5. 点击删除 → 弹出确认 → 调用 API → 成功后刷新列表

## 错误处理

| 场景 | 处理 |
|------|------|
| 网络异常 | Toast 提示「网络异常，请重试」，保留表单数据 |
| 400 参数错误 | 表单字段内联提示错误信息 |
| 409 冲突 | 高亮冲突字段，提示用户修改 |
| 500 服务错误 | Toast 提示「服务异常，请联系管理员」 |
