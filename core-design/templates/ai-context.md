- **技术栈**：Spring Boot 3.x + React + MySQL + Redis
- **核心外部依赖**：HeyGen API v3（数字人视频生成）
- **模块清单**：farvis-video / farvis-avatar / farvis-voice / farvis-credits / farvis-payment / farvis-heygen

### 已上线模块
| 模块 | 状态 | 说明 |

| farvis-avatar | ✅ 已上线 | 数字人形象管理，支持 10 个预设形象 |

## 填写规范

| 区块 | 必填 | 说明 |
|------|------|------|
| 项目概览 | ✅ | 项目基础信息，初始化时填写，变更时更新 |
| 当前状态摘要 | ✅ | 核心区块，每次归档必须更新 |
| 模块索引 | ✅ | 每个模块必须有对应条目，路径必须准确 |
| 迭代历史摘要 | ✅ | 每次归档追加一条记录 |
| 技术债与待优化项 | ⭕ 可选 | 有则记录，无则删除此区块 |
| 同步信息 | ✅ | 每次同步自动更新 |

**篇幅控制**：
- 总字数 ≤ 2000 字（不含代码块标记）
- 摘要类内容一句话描述，不展开细节
- 需要细节时通过索引跳转，不在 AI Context 里堆砌
