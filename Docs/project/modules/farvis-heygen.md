## 1. 模块定位

HeyGen SDK 封装层。对 HeyGen API v3 做统一封装，屏蔽外部 API 的复杂性，对内提供标准化的调用接口。处理代理访问、限流、降级、重试等横切关注点。

---

## 2. 核心业务规则

| # | 规则 | 说明 |
|:-:|------|------|
| R1 | 配额管理 | HeyGen API 1000 次/月，需做限流 |
| R2 | 代理访问 | 国内网络通过代理访问 HeyGen API |
| R3 | 降级策略 | 配额耗尽或 API 不可用时降级处理 |
