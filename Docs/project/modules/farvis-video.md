---

## 1. 模块定位

视频生成核心模块。用户输入脚本 → 选择数字人和声音 → 调用 HeyGen API 合成口播视频。消耗 50 Credits/次。

---

## 2. 核心业务规则

| # | 规则 | 说明 |
|:-:|------|------|
| R1 | 生成消耗 | 每次视频生成消耗 50 Credits |
| R2 | 输出规格 | 1080P 高清输出 |
| R3 | 默认选择 | 默认数字人：杨学瑞，默认声音：雷军演讲 |
| R4 | 状态管理 | 视频状态：生成中 / 已完成 / 失败 |
| R5 | 失败重试 | 生成失败时支持重试，不重复扣费 |
