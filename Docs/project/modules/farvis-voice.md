## 1. 模块定位

语音合成管理模块。管理 5 种公共声音，支持用户上传音频克隆专属语音（消耗 50 Credits/次）。长期方向：200+ 音色储备。

---

## 2. 核心业务规则

| # | 规则 | 说明 |
|:-:|------|------|
| R1 | 公共声音 | v1.0 提供 5 种公共声音 |
| R2 | 语音克隆 | 支持音频上传克隆，消耗 50 Credits |
| R3 | 声音来源 | 区分公共声音和用户私有声音 |
