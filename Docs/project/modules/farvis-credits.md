## 1. 模块定位

积分体系模块。管理用户积分的充值、消耗、流水记录。积分是所有付费操作的统一计量单位。

---

## 2. 核心业务规则

| # | 规则 | 说明 |
|:-:|------|------|
| R1 | 消耗场景 | 视频生成 50C / 数字分身创建 100C / 语音克隆 50C |
| R2 | 充值方式 | 套餐购买（¥99/399/999） |
| R3 | 免费试用 | 免费用户有初始 Credits 用于体验 |
| R4 | 流水记录 | 每笔充值/消耗都记录流水 |
| R5 | 余额校验 | 操作前校验余额是否充足 |
